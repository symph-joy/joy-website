” Joy 可以做什么

  • React 应用

    提供一体的 React 应用解决方案，快速开发页面，管理状态，支持服务端渲染。

  • Node.js 应用

    类似 Spring Boot，开箱即用，提供数据库、缓存、安全等常用组件。（开发中）

  • 混合同构开发

    一体式开发，可一次编写数据结构、接口约定、处理方法等，前后端共同使用。

  • 前后端分离部署

    可一键快速启动整个应用，或者将前端内容编译导出，独立部署和运行。

  • 类型约束

    默认支持和推荐使用TypeScript，提供静态类型检查和高级语法特性。

  • 面向对象

    将面向对象软件开发方法运用到 JS/TS 应用中，能轻松应付复杂的业务场景。

  • 依赖注入

    让代码进一步简单，更容易理解和测试。

// src/cliet/pages/hello.tsx

@ReactController()
export default class Hello extends BaseReactController {
  renderView() {
    return <p>Hello World!</p>;
  }
}

@Controller()
export class HelloController {
  @Get("/hello")
  hello(): string {
    return "Hello World!";
  }
}
